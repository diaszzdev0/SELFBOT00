🔄 Sincronizar Dados

Use esta página para sincronizar os dados entre o painel admin e o painel cliente no celular.

📱 Como usar no celular:
1. Abra esta página no PC e clique em "Gerar Código"
2. Copie o código gerado
3. Abra esta página no celular
4. Cole o código e clique em "Importar Dados"
